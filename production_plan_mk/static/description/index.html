Production planning

Create Production plan




1.After confirming a sale order we can create a production plan against the sale order while clicking 'Production Plan' button. After clicking the button we can see a popup window.







2.Only the 'Planning Manager' can Confirm the plan. We should give access to the user.






3. After save the production plan we can get it from 'Planned' smart button from sale order form. We can also 'Confirm' from there. After confirm the plan manufacturing orders will be created based on the lines.
















4. In the manufacturing order we can see a 'Packing Material Request' button, When clicking the button material request will be created and we can see it from the 'Material Request' smart button.








5. We can set 'Allowed Users' in operation type form. Then the allowed user can only see transfer based on the Operation Type
