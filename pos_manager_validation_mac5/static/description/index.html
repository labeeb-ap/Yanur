POS - Manager Validation

Usage

  1. Go to Settings > Users > Administrator

  2. Click the edit button and select 'Point of Sale' section and enter a pin in the 'Manager Validation PIN' field.

  3. Then save the settings.





  4. Go to Point Of Sale Module.

  5. By clicking the three dots go to settings.





  6. Now scroll down to 'Manager Validation (User)' section and Select the Managers which are responsible for the validation.

  7. Also tick the sections for which the validation is to be done by the Manager.





  8. Then save the settings and Go to POS Dashboard and start 'New Session'.

  9. Now the sections which we already selected for validation in the settings will be asked for a Validation PIN for opening the corresponding section.





  10. Now if the entered PIN is correct you will be able access the section.
