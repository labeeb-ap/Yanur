Pos Counter Closing

Manage Session Closing




1.Added some fields in Payment Methods form.(Pont Of Sale>Configuration>Payment Methods)







2.Closing POS Session using 'Close Session & Post Entries' button









3. After close the session a Counter closing mail will be sent to the selected user. For that we should set mail user in session settings and select closing type on account heads.









Mail format is given below.



4. After that click 'Session Close' wizard and select date click 'Done' button






5. Now we can see new closing form and add required values there and click 'Close' button.






6. After Close a new 'Closing Time' field will be visible and state will be changed to Closed. Now some journal entries will be created based on given details.










7. We can see all closing under Order>Sessions Closed menu.
